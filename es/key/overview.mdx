Enigm Key es un producto físico de emergencia. Contiene conectividad de datos mediante una tarjeta eSIM industrial integrada y se sincroniza con la cuenta Enigm del usuario.

## Overview

Enigm Key permite activar un flujo SOS pulsando tres veces el boton físico. Cuando se activa, notifica a contactos configurados y comparte ubicación en tiempo real hasta que el usuario cancela el envío.

Durante funciónamiento normal, Enigm Key permanece dormido para apoyar la privacidad del usuario.

## Configuration

La vinculacion inicial se realiza desde Enigm App. Los contactos de emergencia tambien se configuran desde Enigm App.

Enigm Command administra Enigm Key cómo dispositivo asociado: visibilidad, estado de ciclo de vida y revocacion.

## Emergency Workflow

El flujo SOS incluye:

- Pulsacion triple del boton.
- Autenticación del dispositivo.
- Comunicacion cifrada con la plataforma.
- Notificación a contactos elegidos.
- Comparticion de ubicación vinculada al evento.
- Cancelacion por el usuario.

## Authentication Model

Cada Enigm Key se autentica con material unico por dispositivo. Las comunicaciones deben estar firmadas, protegidas y cifradas para que la API acepte mensajes del dispositivo.

El modelo está orientado a rechazar tráfico no autenticado, replay attempts y dispositivos no vinculados.

## Privacy Model

Enigm Key no funcióna cómo rastreador rutinario. La ubicación se comparte solo durante el evento de emergencia y hasta que el usuario cancela el envío.

La visibilidad administrativa no debe convertirse en seguimiento permanente ni en acceso a comunicaciones protegidas.

## Revocation And Loss

Enigm Key puede revocarse desde Enigm App y Enigm Command. La revocacion es importante ante perdida, robo, sustitucion o retirada del dispositivo.
