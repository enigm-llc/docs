Esta página consolida la experiencia de dispositivo de Enigm OS: provisionamiento inicial, launcher, Privacy Mode y Controlled Device Management.

## Asistente de configuración

Setup Wizard establece una línea base segura antes de la operación normal.

Flujo previsto:

1. Welcome.
2. Language and Region.
3. Mobile Connectivity.
4. Secure Wi-Fi.
5. Date and Time.
6. Appearance.
7. Strong PIN Setup.
8. Enrolamiento biométrico opcional.
9. Introducción al Modo de privacidad.
10. Términos y privacidad.
11. Inicio de Enigm App.

## Política de PIN fuerte

El onboarding exige autenticación fuerte antes de uso normal. El objetivo es prevenir credenciales debiles y aplicar diseño authentication-first.

## Launcher

El Launcher es la experiencia de inicio orientada a seguridad. Proporciona visibilidad de confianza, estado de red, Privacy Mode, notificaciones de seguridad y acceso directo a Enigm.

El Launcher no calcula confianza; consume resumen de Trust Security Center.

## Modo de privacidad

Privacy Mode permite control rápido sobre sensores sensibles.

Cuando está activo:

- Bloquea camara.
- Bloquea microfono.

No sustituye Device Trust, cifrado de extremo a extremo ni controles del sistema operativo.

## Gestión controlada de dispositivos

Controlled Device Management es opcional. El usuario puede enrolar el dispositivo en modo gestionado para visibilidad de ciclo de vida, estado de seguridad y operaciones remotas mediante Enigm Command.

Remote wipe está disponible solo para dispositivos gestionados enrolados.

## Límites de privacidad

La gestión de dispositivo no proporciona acceso a mensajes, llamadas, multimedia, adjuntos, conversaciones ni claves privadas.

Consulta [Trust Security Center](/es/os/trust-security-center) y [Limitaciones de plataforma](/es/legal/limitations).
