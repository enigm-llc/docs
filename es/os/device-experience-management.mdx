Esta pagina consolida la experiencia de dispositivo de Enigm OS: provisiónamiento inicial, launcher, Privacy Mode y Controlled Device Management.

## Setup Wizard

Setup Wizard establece una línea base segura antes de la operación normal.

Flujo previsto:

1. Welcome.
2. Language and Region.
3. Mobile Connectivity.
4. Secure Wi-Fi.
5. Date and Time.
6. Appearance.
7. Strong PIN Setup.
8. Optional Biometric Enrollment.
9. Privacy Mode Introduction.
10. Terms and Privacy.
11. Launch Enigm App.

## Strong PIN Policy

El onboarding exige autenticación fuerte antes de uso normal. El objetivo es prevenir credenciales debiles y aplicar diseño authentication-first.

## Launcher

El Launcher es la experiencia de inicio orientada a seguridad. Proporciona visibilidad de confianza, estado de red, Privacy Mode, notificaciónes de seguridad y acceso directo a Enigm.

El Launcher no calcula confianza; consume resumen de Trust Security Center.

## Privacy Mode

Privacy Mode permite control rapido sobre sensores sensibles.

Cuando está activo:

- Bloquea camara.
- Bloquea microfono.

No sustituye Device Trust, cifrado de extremo a extremo ni controles del sistema operativo.

## Controlled Device Management

Controlled Device Management es opcional. El usuario puede enrolar el dispositivo en modo gestionado para visibilidad de ciclo de vida, estado de seguridad y operaciónes remotas mediante Enigm Command.

Remote wipe está disponible solo para dispositivos gestionados enrolados.

## Privacy Boundaries

La gestión de dispositivo no proporciona acceso a mensajes, llamadas, multimedia, adjuntos, conversaciones ni claves privadas.

Consulta [Trust Security Center](/es/os/trust-security-center) y [Platform Limitations](/es/legal/limitations).
