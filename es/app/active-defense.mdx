Active Defense es una capacidad de seguridad de Enigm App disponible en produccion para un grupo selecciónado de usuarios.

## Overview

Active Defense analiza comportamiento de red del telefono durante un período acotado solicitado por el usuario desde la app. Su objetivo es identificar patrones compatibles con spyware avanzado o malware móvil.

El análisis se centra en comportamiento de tráfico y señales de red, no en contenido protegido.

## Security Purpose

Active Defense ayuda a detectar riesgos como:

- Comportamiento de red anómalo.
- Patrones compatibles con spyware.
- Indicadores de comunicación sospechosa.
- Riesgo de malware móvil avanzado.

No es intrusivo respecto al contenido: no inspecciona mensajes, llamadas, adjuntos, medios, documentos, conversaciones, claves privadas ni frases de recuperacion.

## User-Controlled Analysis

El usuario inicia el análisis desde Enigm App y puede ver cuando se está ejecutando. El análisis se realiza durante un período definido y no funcióna cómo vigilancia permanente de contenido.

## Processing Model

El análisis se procesa en el entorno de VPN asociado al flujo de Active Defense. Los datos se protegen cifrados durante el procesamiento.

No hay retención de datos de Active Defense. Solo puede conservarse metadata de resultado, cómo la existencia de una detección, cuando el usuario decide mantenerla.

## Relationship With Enigm Intelligence

Active Defense puede aportar contexto de seguridad a Enigm Intelligence y Enigm Command cuando el usuario o la política lo autoriza.

La visibilidad de hallazgos no equivale a visibilidad de mensajes, llamadas o contenido protegido.

## Privacy Considerations

Active Defense está diseñado para apoyar seguridad sin romper privacidad:

- No inspecciona contenido de comunicaciones.
- No accede a claves privadas.
- No crea exportacion de datos de usuario.
- Limita retención a resultado elegido por el usuario.
