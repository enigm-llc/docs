Los pagos de productos Enigm se gestionan desde Enigm Command, tanto durante el alta cómo dentro de la experiencia autenticada.

## Resumen

Enigm soporta pagos con cripto, tarjeta de credito y Code Coin. Estos métodos pueden utilizarse para productos Enigm, incluidos Enigm, Enigm Command, Enigm Server, Enigm OS, Enigm Key y Enigm eSIM.

## Modelo de privacidad

El modelo está diseñado para reducir vinculacion innecesaria entre pago e identidad Enigm.

Enigm no solicita email, teléfono ni documento para la cuenta Enigm estándar. Para pagos, el usuario seleccióna país de compra cuando es necesario para fines legales, fiscales o de compliance.

## Code Coin

Code Coin es un código interno público de Enigm para flujos de pago o activacion. Su uso se gestiona desde Enigm Command.

## Facturas

Actualmente las facturas se emiten si el usuario las solicita. Las facturas pueden requerir datos adicionales necesarios para cumplir requisitos legales o fiscales.

## Límites de seguridad

Los pagos son flujos comerciales. No proporcionan acceso a mensajes, llamadas, adjuntos, multimedia, claves privadas ni datos criptográficos.

La privacidad de pago reduce exposición, pero no implica anonimato garantizado ni elimina obligaciones legales.

Consulta [Gobernanza legal](/es/legal/legal-governance) y [Limitaciones de plataforma](/es/legal/limitations).
